Text Style

The website uses the following text styles:

H1: Main Page Heading

  • Font-weight: 700 (bold)
  • Font-size: 26px
  • Font-Family: Open Sans

H2: Sub-Heading

  • Font-weight: 500
  • Font-size: 20px
  • Font-Family: Montserrat

P: Paragraph Text

  • Font-weight: 400 (regular)
  • Font-size: 16px
  • Font-Family: Source Sans Pro
